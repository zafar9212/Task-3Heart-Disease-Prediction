# Task-3Heart-Disease-Prediction
#  Heart Disease Prediction using Machine Learning

##  Project Overview

This project predicts whether a patient has heart disease based on medical attributes.

It is a binary classification problem:
- 0 → No Heart Disease
- 1 → Heart Disease Present

The project demonstrates a complete ML workflow including data preprocessing, model training, and evaluation.

---

##  Dataset Information

The dataset contains medical features such as:

- Age
- Sex
- Chest pain type
- Resting blood pressure
- Cholesterol
- Maximum heart rate
- Exercise induced angina
- And other clinical measurements

Target column:
- `target` (0 or 1)

---

##  Workflow

1. Import required libraries  
2. Load dataset  
3. Exploratory Data Analysis (EDA)  
4. Feature and target separation  
5. Train-test split (80/20)  
6. Logistic Regression model training  
7. Model evaluation  

---

##  Model Used

### Logistic Regression

Why?
- Suitable for binary classification
- Efficient and interpretable
- Commonly used in medical prediction problems

---

##  Model Evaluation

Metrics Used:

- Accuracy Score
- Confusion Matrix
- Precision
- Recall
- F1-score

These metrics evaluate how well the model predicts heart disease cases.
